# DemoWebShop Automation Framework

## Overview

This project is an automated testing framework developed for the Demo Web Shop application using Selenium WebDriver, Java, TestNG, and Maven.

The framework follows the Page Object Model (POM) design pattern and automates core e-commerce functionalities including Login, Product Search, Shopping Cart, Checkout, and Order Validation.

## Technologies Used

* Java
* Selenium WebDriver
* TestNG
* Maven
* Page Object Model (POM)
* PageFactory
* Data-Driven Testing (DDT)
* Eclipse IDE

## Framework Features

* Page Object Model implementation
* PageFactory for WebElement initialization
* Reusable Base Class for browser setup
* TestNG test execution
* Maven dependency management
* Data-Driven Testing using external test data
* Modular and maintainable framework structure
